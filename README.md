# 🏝️ Treasure Island Game

A fun text-based adventure game built using Python where the player makes choices to find hidden treasure.

## 💡 Features
- Interactive story-based gameplay
- Multiple decision paths
- Different outcomes based on user choices
- Beginner-friendly command-line game

## 🛠️ Tech Used
- Python

## ▶️ How to Run
1. Install Python
2. Run the file:
   python treasure_island.py

## 📌 Example
Welcome to Treasure Island.
Your mission is to find the treasure.

You're at a crossroad. Where do you want to go? Left or Right
Left
You come to a lake. Do you want to wait for a boat or swim?
Wait
You arrive at the island unharmed. There is a house with 3 doors.
Which colour do you choose? Red, Blue, or Yellow
Yellow
You found the treasure! You Win!
